# 📊 Crypto Data Analysis & AI Trading

Este repositorio documenta mi proceso de aprendizaje y aplicación práctica en el análisis de datos de criptomonedas utilizando Python, APIs financieras y técnicas de Machine Learning orientadas al trading.

## 🚀 Descripción

A lo largo de este proyecto trabajé en la integración de datos de criptomonedas mediante APIs, el procesamiento de información financiera y la implementación de distintos modelos analíticos para mejorar la toma de decisiones en trading.

El enfoque principal es explorar cómo la ciencia de datos y la inteligencia artificial pueden complementar el análisis técnico tradicional.

---

## 🧠 Temas Cubiertos

* Consumo de APIs de criptomonedas
* Manipulación y limpieza de datos con Python
* Análisis exploratorio de datos (EDA)
* Visualización de datos financieros
* Indicadores técnicos
* Backtesting de estrategias
* Modelos de Machine Learning aplicados al trading:

  * Regresión Lineal
  * Random Forest
  * Clustering

---

## 🛠️ Tecnologías Utilizadas

* Python
* Pandas / NumPy
* Matplotlib / Seaborn
* Scikit-learn
* APIs de criptomonedas
* Jupyter Notebook

---

## 📈 Objetivo

Desarrollar una base sólida en el uso de herramientas de análisis de datos y modelos predictivos para:

* Mejorar el análisis técnico en trading
* Identificar patrones en el mercado de criptomonedas
* Evaluar estrategias mediante backtesting
* Comprender el comportamiento de los datos financieros

---

## 📂 Estructura del Repositorio

El repositorio está organizado por sesiones/clases, donde cada sección incluye:

* Notebooks con ejemplos prácticos
* Scripts de análisis
* Implementaciones de modelos
* Ejercicios y pruebas de estrategias
